项目经历

4 个企业级 AI 应用项目

GoWhere 智能旅行助手

Spring Boot 3 Spring AI RAG ReAct Agent MCP Vue 3

基于 Spring Boot 3 + Spring AI + RAG + Tool Calling + MCP 的企业级旅行助手，为用户提供旅行指导服务。系统支持多轮对话、记忆持久化、RAG 知识库检索等能力，并基于 ReAct 模式实现 Agent 自主推理与工具调用。通过集成联网搜索、网页抓取、MCP 图片检索、天气查询及 PDF 自动化生成工具，能够高效完成复杂的个性化旅行定制计划。

  • •从零实现 ReAct Agent 四层架构，手动管理多步工具调用上下文，参考 OpenManus 设计
  • •RAG 全链路调优：查询重写 + 关键词元数据增强，对比三种向量存储方案（SimpleVectorStore / PgVector / 阿里云知识库）
  • •自研 FileBasedChatMemory，Kryo 高性能序列化持久化对话历史，服务重启后可恢复

SmartOrder 智能点单系统

Google ADK A2A Protocol FastAPI React 18 Prometheus

SmartOrder 是基于 FastAPI + Google SDK + A2A 的智能点单系统。用户输入自然语言描述，终端智能体通过标准的 A2A 协议与远程点单智能体、配送智能体进行协同，实现从 AI 对话、商品下单、配送的全自动流水线。

  • •基于 Google A2A 协议实现多 Agent 分层架构，Root Agent 根据用户意图自动路由至领域 Agent
  • •Agent 通过 HTTP 调用后端 REST API，与业务逻辑完全解耦，支持独立部署与迭代
  • •集成 Prometheus + Grafana，监控 QPS / P95 延迟 / 5xx 错误率，预置 Dashboard 开箱即用

GenApp 零代码生成平台

LangChain4j LangGraph4j Spring Boot 3 Java 21 Redis

GenApp 是基于 Spring Boot 3 + LangChain4j + LangGraph4j 的 AI 零代码应用生成平台。用户输入自然语言需求，由 AI Agent 自动执行并发图片搜集、代码生成、质量检查、项目构建的完整工作流，最终一键部署为可访问的 Web 应用。

  • •LangGraph4j 并行工作流，图片搜集耗时缩短 75%；支持 HTML / Vue 多框架差异化生成
  • •Redis + Caffeine 双级缓存；虚拟线程异步处理 I/O 密集型任务，规避线程阻塞
  • •Redisson RRateLimiter + AOP 实现用户 / IP 级限流；LangChain4j 输入护轨防注入

HotMonitor 热点监控工具

Vibe Coding Agent Skills MCP WebSocket GitHub Copilot

基于 Vibe Coding 开发实现，帮助用户第一时间发现和跟踪热点，支持用户自定义监控关键词。系统通过 Twitter 等 8+ 数据源聚合采集信息，利用 AI 大模型进行真伪辨别、相关性分析、摘要生成，并通过 WebSocket 实时推送和邮件通知。

  • •全程 Vibe Coding，AI 辅助整体效率提升约 3 倍；8+ 数据源聚合采集 Twitter 等平台
  • •AI 自动将关键词扩展为 5-15 个语义变体，信息召回率相比精确匹配提升 300%+
  • •封装为完全自包含的 Agent Skills 技能包，无需后端即可在 Cursor / VS Code / Claude Code 中使用

学术论文与探索性项目

CCF-A IEEE TMC

Channel-Triggered Backdoor Attack on Semantic Communication Systems

后门攻击 语义通信 AI 安全 无线信道

本文提出了一种新的用于语义通信系统的信道触发后门攻击（CT-BA）框架，使用固有的无线信道特性（例如，衰落，噪声）作为隐形触发器，而不是输入修改。

  • •利用无线信道固有特性作为后门触发器，增强隐身性，绕过常规防御机制
  • •通过通道增益和噪声功率谱密度的解耦设计，实现多维触发器可配置
  • •无线信道的开放性和时变特性为对手提供巨大操作灵活性，即使无主动参与也能触发
3D-RAG 探索性项目

AgentRay — 3D 电磁场可视化与 AI 语义分析

Vue 3 Three.js FastAPI Qwen-VL ChromaDB

Vue 3 + Three.js/TresJS + FastAPI + ChromaDB + Qwen-VL 空间 AI 可视化系统。将 RAG 范式扩展至三维空间数据，通过 CV 特征提取和多模态 LLM 实现空间语义理解。

  • •"3D-RAG"：基于 CV 特征提取（建筑覆盖率、熵、角点密度、天际线粗糙度）扩展传统 RAG
  • •多模态 LLM（Qwen-VL + Qwen-Max）实现语义空间理解与自然语言交互
  • •asyncio.gather() 并发优化 + InstancedMesh + LOD 策略高效渲染 10k+ 点云
